# Specification

## Summary
**Goal:** Create the initial InstaBhai App with a landing page, basic backend health check, and a vibrant Indian-inspired design theme.

**Planned changes:**
- Create a landing page with a welcome message and basic navigation structure
- Set up a backend actor with a health check method that returns application status
- Implement a modern design theme with warm colors (orange, red, yellow accents), bold typography, and energetic layouts inspired by Indian aesthetics

**User-visible outcome:** Users can access a welcoming landing page with vibrant Indian-inspired design, see a prominent welcome message, and navigate through a basic structure. The frontend successfully connects to the backend health check.
